    活动规则

    1. 活动期内，奇幻城国际进行存款的存款额进行排名。

    2. 此活动需在活动时间内存款100以上无需申请，系统将自动报名参予。活动结束后，以奇幻城存款王排名数据为准，奇幻城将依名次送出礼品或彩金。

    3. 同一玩家、同一信息只能获得一个奖品。

    4. 前40名获得实物奖品的玩家，若放弃奖品折换成彩金，彩金是官方价的70%，无需流水即可提款。

    5. 此次优惠每位玩家﹑每户﹑每一住址 、每一电子邮箱地址﹑每一电话号码﹑相同支付方式(相同借记卡/信用卡/银行帐户姓名及号码) 及IP地址只能享有一次优惠,一旦确认为套利玩家，立即没收盈利和本金。

    6. 一旦奇幻城国际发现客户有滥用此促销的行为，我们保留撤回此奖金及由此赢得的任何彩金的权力。一旦此种情况发生，奇幻城国际将终止玩家享受此优惠的资格。

    7. 奇幻城对本次优惠享有最终解释权。
