    2.彩金将会在比赛结束当天18:00之前派发完成。

    3.获胜的公会按投注占比获得奖金，奖金最低金额为8元。

    例如获胜公会中每位玩家将可得到（投注占比）*76万的奖金

    投注占比=玩家投注额/所属公会总投注* 100%

    例如您所属的A公会总投注为：100万

    您的投注额为：77万

    您的投注占比即为：77/ 100 * 100% = 77%

    您可获得 77% *76万元 = 585200元奖金

    1.彩金无需流水可提款。
