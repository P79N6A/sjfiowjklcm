    活动内容

    计算每个龙舟玩家的流水总和（仅限DT老虎机），按最高金额的龙舟队进行排名可以获得相应的奖金。

    此次龙舟流水赛活动奖金为10万元，奖金将由流水最高之龙舟获得。

    获胜龙舟中每位玩家将可得到（投注占比）*10万的奖金

    投注占比=玩家投注额/所属龙舟总投注* 100%

    例如您所属的A龙舟总投注为：100万

    您的投注额为：10万

    您的投注占比即为：10 / 100 * 100% = 10%

    您可获得 10% * 10万 = 1万元 奖金

    如何参与

    每艘龙舟可乘载 300名会员，达到300名，将会无法再选取。

    活动开始将会有三组龙舟进行选取，如果三艘龙舟都满员，我们将会再开启一艘龙舟。

    请在5月22日之前在优惠页面进行点取”我要参加”进行报名选取龙舟，并在2017年5月22日00:00:00 — 2017年5月29日23:59:59投注DT平台进行龙舟流水竞赛。

    彩金如何派发

    彩金将会在5月29日23:59分后开始结算，并在5月30日下午18:00之前进行派发完成。

    获胜的龙舟队按投注占比获得奖金，奖金最低金额为8元 。（注：活动期间需要在DT有最低1元的投注金额）

    活动规则

    1.龙舟选择需要在5月22日之前进行选择，否则视同放弃。

    2.彩金无需流水可提款。

    3.每个会员 只能拥有一个账号参加此活动，包括电话、邮箱、IP、地址、支付方式（相 同借 记卡/信用卡/银行账户号码），使用多个账户我们将有权取消会员申领资格。

    4.有效投注计算时间为2017年5月22日00:00:00 -- 2017年5月29日23:59:59，活动结束将会以站内信派发及公告告知。

    5.奇幻城国际享有最终解释权。
